## üé¨ RECORDING TIPS

### Before You Hit Record:
1. Do a **dry run** - Practice the entire flow once
2. Close all notifications (Do Not Disturb mode)
3. Clean up desktop if you'll show it
4. Have the script visible on another screen or printed
5. Test your recording setup with a 30-second clip

### During Recording:
1. **Slow down** - Move mouse slowly, pause between actions
2. **Hold on important screens** - OAuth consent (5-7 sec), code (3-5 sec)
3. **Zoom in when needed** - Especially for code sections
4. **Don't rush** - Better to have a 7-minute video than miss important details
5. **Pause between scenes** - Makes editing easier

### After Recording:
1. Watch the entire video
2. Check that OAuth consent screen was captured clearly
3. Verify all 4 scopes are visible
4. Confirm DeepSeek blocking is shown
5. Make sure audio is clear (if recorded live)
