## üé• RECORDING TIPS

### Before Recording:
1. **Clear browser cache/cookies** - This ensures OAuth screen appears fresh
2. **Revoke app permissions** - Go to https://myaccount.google.com/permissions
3. **Prepare test data** - Have a few test emails ready in Gmail
4. **Test API key** - Have a real or fake OpenRouter API key ready to paste
5. **Close unnecessary windows/tabs** - Clean desktop for recording

### During Recording:
1. **Record in 1080p (1920x1080)** - Minimum resolution for Google's review
2. **Use full screen mode** - Hide browser toolbars if possible
3. **Slow mouse movements** - Give viewers time to see what you're clicking
4. **Pause on important screens** - Hold on OAuth consent for 3-5 seconds
5. **Zoom in on code** - If showing code, increase font size (Cmd/Ctrl +)

### Recording Tools:
- **QuickTime** (Mac): Built-in, simple, good quality
- **Loom**: Free, easy to use, auto-uploads
- **OBS Studio**: Free, more control, best quality
- **Screen Studio** (Mac): Professional, paid, best visual polish

### Voiceover:
- Use the updated `DEMO_VIDEO_SCRIPT_ELEVENLABS.txt`
- Upload to ElevenLabs for AI voiceover
- Or record your own voiceover with good microphone
- Add voiceover in post-production (iMovie, DaVinci Resolve, etc.)
